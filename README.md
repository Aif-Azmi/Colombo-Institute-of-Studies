Colombo Institute of Studies - Employee Management System | Java OOP + MySQL 🏆 

Developed a complete GUI-based Employee Management System for the Colombo Institute of Studies, replacing their manual system with an efficient object-oriented solution backed by MySQL database. This project received a Distinction-level grade (>70%) for demonstrating excellence in OOP principles and database integration.

The Challenge: Design and build a digital system to manage employee records, departments, designations, and user access for an educational institute currently operating with manual processes.

Object-Oriented Concepts Implemented:
• Inheritance: Class hierarchies with HRManager, HR Assistant, and Admin subclasses extending User class
• Encapsulation: Private attributes with controlled access through setter/getter methods (e.g., Employee class)
• Polymorphism: Method overriding and overloading for flexible operations
• Abstraction: Simplified interfaces hiding complex business logic
• Vector Class: Dynamic resizable container for storing and managing employee objects

System Features:
• Role-Based Access: Separate dashboards for HR Managers, HR Assistants, and Administrators
• HR Manager Functions: Add/update departments, designations, employees; search by name, EPF number, department, or designation
• Admin Functions: Create and manage user accounts for HR Managers and Assistants with secure access control
• HR Assistant View: Employee search capabilities with appropriate access restrictions
• Data Integrity: Encapsulation ensures secure data handling with MySQL persistence

UML Design:
• Use Case Diagram: Complete actor-system interactions showing all functionalities
• Class Diagram: Comprehensive structure with relationships between Employee, Department, Designation, User, and subclasses
• Sequence Diagrams: Detailed interaction flows for HR Manager, HR Assistant, and Admin operations

Technical Stack:
• Language: Java with Swing framework
• GUI Components: JFrame-based windows for all interfaces
• Database: MySQL for persistent data storage
• Data Management: Vector collections with database synchronization

Academic Recognition: Awarded distinction for comprehensive OOP implementation, clean code structure, professional UML modeling, MySQL integration, and effective system design that meets real institutional needs.
